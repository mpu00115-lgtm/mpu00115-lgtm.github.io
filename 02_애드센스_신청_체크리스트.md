## ⏳ 신청 전 남은 일 (순서대로)

### 1단계: 색인 (지금 진행 중)
- [ ] Search Console에서 sitemap.xml "성공" 확인 (며칠 소요)
- [ ] `site:mpu00115-lgtm.github.io` 검색 시 페이지들이 뜨는지 확인
- [ ] 주요 페이지 5~6개 "색인 생성 요청" (URL 검사 도구)

> 색인이 어느 정도 된 뒤 신청해야 승인율이 높습니다. (최소 며칠~2주)

### 2단계: 애드센스 신청
1. https://adsense.google.com → 가입 (mpu00115@gmail.com)
2. 사이트 주소 입력: `mpu00115-lgtm.github.io`
3. 애드센스가 주는 **`<script>` 코드 1줄**을 받음
4. 그 코드를 **모든 페이지의 `<head>`** 안에 붙여넣기
   → 이건 자동 일괄 삽입 도와드림 (지금은 pub-ID를 몰라서 대기)
5. GitHub 재업로드 → 애드센스에서 "검토 요청"

### 3단계: 승인 후
- [ ] `ads.txt`의 `pub-0000...`을 **실제 pub-ID**로 교체
- [ ] 계산기 페이지의 `[ AdSense — ... ]` 자리에 실제 광고 단위 코드 삽입
  - 자리는 이미 만들어둠: 각 페이지 "Above the fold" 1개 + "In-content" 1개
