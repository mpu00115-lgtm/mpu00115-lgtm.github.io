# 애드센스 승인 설계도 (2026 요건 기반)

> 거절 1위 사유 = **필수 페이지 누락 + 저가치 콘텐츠**. 이 설계는 그 둘을 정면으로 해결한다.
> 작성 2026-06-07. 출처: Google AdSense Help(공식), Mozedia, Webtimize, Termly 등 2026 자료.

## 1. 2026 승인 요건 (공식 + 실측 종합)

| 요건 | 기준 | 우리 대응 |
|---|---|---|
| **필수 신뢰 페이지** | About, Contact, Privacy Policy, Terms (누락=거절 1위) | ✅ **4종 다 만듦**(이 폴더) |
| **콘텐츠 양** | 고품질 원본 글 **15~30편**(최소 10~15), 각 **800~1,200+단어** | 지원글 15편 + 계산기 페이지 |
| **콘텐츠 질** | 사용자 쿼리를 완전히 답하는 깊이. 300단어 얇은글 ❌ | 계산기+해설 400단어+ / 가이드 1,000단어+ |
| **도메인 연령** | 1개월+ (이상적 3~6개월) | 도메인 먼저 사두고 콘텐츠 쌓기 |
| **트래픽** | 공식 요건 없음, 일 10~50명이면 유리 | SEO + (유튜브 유입) |
| **기술** | HTTPS, 모바일 친화, 깔끔한 내비 | ✅ Cloudflare(HTTPS자동)+반응형 |
| **이메일** | 도메인 이메일 권장(you@yourdomain.com) | 도메인 메일 설정 |
| **정책 준수** | 금지 콘텐츠 없음, 원본 | YMYL 면책 필수 |

**승인 소요**: 모든 요건 충족 시 평균 **3~5일**.

## 2. 승인용 사이트 구조 (최소 구성)

```
yourdomain.com/
├── index.html              ← 홈(도구 목록 + 사이트 소개)
├── about.html        ★필수  ← 운영자·미션(실존 증명)
├── contact.html      ★필수  ← 연락 수단(폼 or 도메인 이메일)
├── privacy.html      ★필수  ← AdSense 쿠키·3자광고 고지(가장 중요)
├── terms.html        ★필수  ← 이용약관 + 면책
│
├── calculators/            ← 도구 페이지(인터랙티브=가치신호)
│   ├── mortgage-calculator.html   ✅이미 만듦
│   ├── auto-loan-calculator.html
│   └── ...
│
└── guides/                 ← 지원 글 15편(800~1,200단어, 승인 핵심)
    ├── how-to-lower-mortgage-payment.html
    ├── fixed-vs-adjustable-rate.html
    └── ...
```

**승인 최소 묶음**: 필수페이지 4 + 계산기 3~5 + 지원글 12~15 = **약 20페이지**.

## 3. 콘텐츠 계획 (지원글 15편 — 승인을 가르는 부분)

순수 계산기만 = 거절. 각 계산기 주변에 **검색의도를 채우는 가이드 글**을 배치한다. 모기지 니치 예시:

## 4. 승인 전 최종 체크리스트

**페이지**
- [ ] About/Contact/Privacy/Terms 4종 라이브 + 푸터에서 모든 페이지 링크
- [ ] Privacy에 **"Google AdSense·3자 쿠키·DoubleClick" 고지** 포함(필수)
- [ ] Contact에 실제 도메인 이메일 or 작동 폼

**콘텐츠**
- [ ] 지원글 12~15편(각 800단어+), 원본·검토 완료
- [ ] 계산기마다 해설 400단어+ + FAQ + 면책
- [ ] 복사·중복 콘텐츠 0

**기술**
- [ ] HTTPS(Cloudflare 자동), 모바일 반응형, 깨진 링크 0
- [ ] 빠른 로딩(Lighthouse 90+), 사이트맵 + GSC 제출
- [ ] 도메인 1개월+ 연령

**정책**
- [ ] 금지 콘텐츠 없음, YMYL 면책 명시
- [ ] 18세+ 본인 계정, 도메인 이메일

→ 위를 다 채우면 신청. **처음부터 500페이지로 신청 금지** — 20~30페이지 품질로 선승인 후 확장.
